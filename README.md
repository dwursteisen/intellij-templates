IntelliJ 12 code templates
========


Sample
-------

Ctrl+J them type "that" to get


How to get it ?
-------

	Close IntelliJ
	git clone xxxx templates # into your IntelliJ preference folder
	relaunch IntelliJ

IntelliJ preferences folder: 
-----
On Mac OS X
	~/Library/Preferences/IntelliJIdea12
